%% README

The script plsc_script.m performs partial least squares correlation analyses. The methods are described in detail
in Griffis et al., 2019 (Cell Reports).

The Support_Functions folder and subfolders contain necessary functions,
and must be in your MATLAB path to run the script.

All code is written by me with the exception of the paq.m and eigen.m functions,
which were obtained from Herv Abdi's website.

Joseph Griffis, 2019, Washington University in St Louis
